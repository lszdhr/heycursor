# HeyCursor

底部面板中的 Messenger，**显示名为 HeyCursor**，扩展 ID 为 **`local.heycursor`**。

- 可与侧栏 **Cursor Messenger**（`local.cursor-mcp-messenger`）**同时安装**，互不覆盖。
- MCP 仍注册为 **「Cursor Messenger」**，与侧栏版 **共用**；任一侧执行一次「安装 MCP」即可。
- 输入草稿使用独立键 `heycursorWebviewState`，与侧栏版草稿不混用。

安装后在底部面板选择 **HeyCursor** Tab（与终端、端口同一区域）。
